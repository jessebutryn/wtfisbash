# alias

This is a collection of some of the aliases I use.

## Because I came from microsoft

```bash
alias cls='clear'
alias cd..='cd ..'
```

## macos

```bash
alias yum='brew'
alias ls='/usr/local/bin/gls --color'
```

## Miscellaneous

```bash
alias ll='ls -lahF'
```
